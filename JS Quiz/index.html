JavaScript Array Exercises

Adapted from from http://www.w3resource.com/javascript-exercises/javascript-array-exercises.php

1) Check whether a given value is an array or not. Should return `true` or `false`.

2) Clone an array, and compare the new array to the original. The function should return `true` if the cloned array is actually a different object (which it should be).

3) Get the first element of an array. Passing a second argument 'n' will return the first 'n' elements of the array.

4) Get the last element of an array. Passing an argument 'n' will return the last 'n' elements of the array.

5) Join elements of an array into a single string with a given expression.

6) Write a function which accept a numeric string as input and insert dashes (-) between each two even numbers. For example if you accept 025468 the output should be 0-254-6-8.

7) Sort an array of numbers from lowest to highest.

8) Return the most common element in an array. If tied, return the element that appears first in the array.

9) Swap the case of each character of a string. For example if you input 'The Quick Brown Fox' the output should be 'tHE qUICK bROWN fOX'.

10) Remove duplicate items from an array, ignoring case. That is, duplicates should be removed even if case does not match.

11) Find the leap years within a given range of years.

12) Given two arrays of numeric values, sum together each individual index value. e.g. [1, 2, 4] + [0, 5] = [1, 7, 4]

13) Return duplicate values in an array, i.e. any value that appears more than once.

14) Flatten a nested array of any depth. e.g. [1, [[2], 3, 4]] becomes [1, 2, 3, 4]

15) Compute the union of two arrays, eliminating duplicates.
